This package implements the configuration mechanisms.
The XML configuration is contained in XML files searched in the class path.

The XML configuration is accessed using XPath strings and is based on Xalan/Jaxen implementation.

To access the configuration parameters it is available the XMLConfig class that exposes many static methods.

For example: 

        // reads all nodes that match the given Xpath
        //
        NodeList nodeList = XMLConfig.getNodeList(XMLFILE, XPATH);

        // reads a node value
        //
        String value = XMLConfig.get(XMLFILE, XPATH);

To use the result you must import the org.w3c.dom.* package.

Note that the files are readed only the first time they are accessed. The files are not checked for changes.
To reload all files you must call the XMLConfigu.reloadAll() method (note that this method does not actually reload the configuration files, they will be reloaded when requested).
